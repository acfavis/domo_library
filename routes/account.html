get_accounts

 get_accounts (auth:domolibrary.client.DomoAuth.DomoAuth,
               debug_api:bool=False,
               session:Optional[httpx.AsyncClient]=None)

retrieve a list of all the accounts the user has read access to. Note users with “Manage all accounts” will retrieve all account objects

sample implementation of get_accounts

import os
import pandas as pd

token_auth = dmda.DomoTokenAuth(
    domo_instance="domo-dojo",
    domo_access_token=os.environ["DOMO_DOJO_ACCESS_TOKEN"]
)


res = await get_accounts(auth = token_auth)

GetAccount_NoMatch

 GetAccount_NoMatch (account_id, domo_instance, status,
                     function_name='get_account_from_id')

base exception

sample implementation of get_account_from_id

import os
import pandas as pd

token_auth = dmda.DomoTokenAuth(
    domo_instance="domo-dojo",
    domo_access_token=os.environ["DOMO_DOJO_ACCESS_TOKEN"]
)
try:
    res = await get_account_from_id(auth=token_auth, account_id = 152)
    res.response
except GetAccount_NoMatch as e:
    print(e)
get_account_from_id: Status 403 - account_id 152 not found at domo-dojo
